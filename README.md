# Space Invaders Game

Welcome to the Space Invaders project, a Python application built using Pygame that recreates the classic arcade experience. Control your spaceship, shoot down waves of enemies, and try to achieve the highest score.

## Features

* Classic Space Invaders-style gameplay
* Player movement and shooting mechanics
* Enemy waves with increasing difficulty
* Score tracking system

## Prerequisites

Before running the Space Invaders game, ensure you have the following installed:

* Python (3.8 or higher recommended)
* pip (Python package manager)
* Pygame library

## Installation

Follow these steps to get your Space Invaders game up and running:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/imtiazhasan07/space-invaders.git
   cd space-invaders
   ```

2. **Install dependencies:**
   Install Pygame using pip:

   ```bash
   pip install pygame
   ```

## Usage

To launch the game, run the following command from the project directory:

```bash
python main.py
```

(Replace `main.py` with your actual main file name if different.)

Once the game starts, use your keyboard to control the player and begin playing.
